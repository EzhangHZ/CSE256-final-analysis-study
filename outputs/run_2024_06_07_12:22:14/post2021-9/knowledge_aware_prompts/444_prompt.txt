# Knowledge-Aware Prompt:

1. Read t, the number of test cases.

2. Loop t times to read each test case.

3. Read n, the size of the array.

4. Read the elements of the array and store them in a list.

5. For each element in the array, repeatedly apply the operation until all array elements are equal.

6. Check if it is possible to make all array elements equal. If all elements are equal, print "YES"; otherwise, print "NO".

7. End loop.
