# Knowledge-Aware Prompt:

1. Read t, the number of test cases.

2. Loop t times to read each test case.

3. Read the string s for each test case.

4. Check if the length of s is odd. If it is, print "NO" and move on to the next test case.

5. Construct a new string by concatenating s with itself.

6. Check if the concatenated string contains s. If it does, print "YES"; otherwise, print "NO".

7. End all loops.
